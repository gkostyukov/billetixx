# claws

Локальная папка для моих служебных скриптов/утилит для работы с OANDA API и диагностики.

## Структура
- `profiles/` — сюда ты можешь скинуть твой профайл (ключи/настройки) для **локального** использования.
- `scripts/` — скрипты, которые используют профайлы и дергают API.

## Важно
- Эта папка **не должна** попадать в публичный репозиторий вместе с ключами.
- Профайлы лучше хранить в файлах, которые добавлены в `.gitignore`.

## Как будет работать
План: скрипты будут брать активный профайл из `profiles/<name>.json` и создавать OANDA-клиент.

Ты пришли, пожалуйста:
- какие поля ты хочешь в профайле (practice/live, accountId, apiToken)
- или просто пришли свой профайл — я подстрою скрипты под формат.
